# 概要
* シンプルな操作で毎日の支出を見える化できるデザイン
* 入力した情報がカレンダーに表示されます。
* カーソルを合わせると支出の内容が詳しくわかります。
* 一覧画面では円グラフでその月に使った費用の内訳を知ることができます。
